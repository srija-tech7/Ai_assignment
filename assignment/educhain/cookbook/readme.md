## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/satvik314/educhain
   ```
2. Navigate to the folder:
   ```bash
   cd educhain/cookbook
   ```
3. Open the desired `.ipynb` file in your Jupyter Notebook environment and start creating!
